Inventário

Este aplicativo foi desenvolvido com objetivo de auxiliar o desenvolver na realização de inventários em seu local de trabalho.

O aplicativo foi desenvolvido para uso diário e temporário. Os dados salvos no aplicativo podem ser perdidos quando o usuário limpar os dados do navegador WEB do android. Portanto, esteja ciente que os dados salvos não são permanentes, isto é, não estão salvos na nuvem.
